A simple, clean Next.js + Tailwind site with:
- Home hero headline: "You're not the problem, the hat was."
- Nav tabs: Products / About Us / Contact Us
- Subtle background texture overlay
- Products grid (placeholder images included)

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Add your hat images
Replace files in `public/products/` with your real images (keep filenames or update `src/lib/products.ts`).

## Deploy
Import the repo into Vercel and deploy. Then connect your domain in Vercel settings.
